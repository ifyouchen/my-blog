# My-Blog 产品说明文档

> 版本：1.0.0 · 更新时间：2026-05-13

---

## 一、产品概述

My-Blog 是一个面向技术人群的**全功能博客与知识社区平台**，集内容创作、互动社区、学习路径、实时通讯、推荐引擎于一体。平台支持个人技术文章发布、专栏/专题体系构建、读者互动（评论/点赞/收藏/关注）、搜索发现，以及一套完整的后台运营体系。

---

## 二、目标用户

| 用户角色 | 使用场景 |
|----------|----------|
| **内容创作者** | 发布技术文章、管理专栏、查看创作数据 |
| **读者** | 浏览文章、参与评论、学习专题课程、关注博主 |
| **运营管理员** | 内容审核、用户管理、广告投放、数据监控 |

## 三、核心功能模块

### 3.1 用户体系

#### 3.1.1 注册与登录
- **注册方式**：用户名 + 邮箱 + 密码，支持**邀请码**和**邮箱验证码**双重校验
- **登录方式**：用户名或邮箱 + 密码登录，返回 JWT Token
- **密码找回**：通过注册邮箱接收重置链接，Token 有效期 30 分钟
- **账号安全**：登录时记录 IP 和时间戳；账号被管理员禁用后立即拦截（即使 Token 未过期）

#### 3.1.2 个人主页与资料
- **基础资料**：昵称、头像（支持上传）、个人简介
- **扩展资料**：个人网站、GitHub、Twitter/X、所在地
- **社交统计**：文章数、总浏览数、总点赞数、粉丝数、关注数
- **修改限制**：密码修改需验证旧密码；邮箱更换需短信/邮件验证码

#### 3.1.3 关注体系
- 互相关注即形成粉丝/关注关系
- 可查看粉丝列表、关注列表（分页）
- 检查与任意用户的关注状态
- 关注成功后被关注者实时收到**站内通知**

### 3.2 内容创作

#### 3.2.1 文章生命周期

```
草稿(DRAFT) ──────────────────► 已发布(PUBLISHED)
     │                                  │
     ├─► 定时发布(SCHEDULED) ──────────►│
     │                                  │
     └──────────────────────────────► 下架(OFFLINE)
                                        │
                                        ▼
                                     软删除(DELETED)
```

- **草稿**：随时保存，无格式校验
- **发布**：要求标题、正文、分类均不为空；发布前自动扫描敏感词并标记 `warnFlag`
- **定时发布**：指定未来发布时间，后台定时任务自动触发
- **下架**：支持填写下架原因（如违规/过期内容）
- **精选**：管理员可将文章设为精选，支持权重排序（0-1,000,000）

#### 3.2.2 文章内容管理
- **富文本内容**：支持长文档（正文字段 mediumtext）
- **封面图片**：可上传或使用系统默认封面
- **标签**：自由输入标签名，系统自动同步维护标签统计
- **分类**：选择预设分类
- **SEO 优化**：独立设置 SEO 标题、SEO 描述、URL slug（唯一标识）
- **版本历史**：每次发布/重大更新自动保存版本快照，支持回滚到任意历史版本

#### 3.2.3 创作数据仪表板
- **概览卡片**：总文章数、总浏览量、总点赞数、总收藏数
- **趋势图表**：日/周/月 维度的文章发布趋势、阅读趋势
- **文章表现**：单篇文章浏览、点赞、收藏、评论数排名

### 3.3 内容发现

#### 3.3.1 文章列表与筛选
- 支持按**分类**、**标签**、**关键词**、**状态**筛选
- 支持按**发布时间**、**热度**（阅读+点赞综合）、**最新**排序
- 分页加载，默认每页 20 条

#### 3.3.2 推荐系统
- **登录用户**：基于兴趣标签 + 关注作者加权推荐；过滤已读/已点赞文章
- **匿名用户**：基于全站热度（精选权重 + 点赞量 + 阅读量）
- 推荐结果缓存 1 小时，兼顾实时性与性能

#### 3.3.3 排行榜
- **文章排行**：按热度/点赞量排名
- **作者排行**：展示粉丝数、文章数、总阅读量、总点赞量，附带作者最热文章

#### 3.3.4 全文搜索
- 跨**文章**（标题/摘要/正文）、**专题**、**专栏**、**标签** 统一搜索
- 关键词高亮展示
- 热门关键词展示（缓存 5 分钟）
- 登录用户自动保存搜索历史，可管理删除

### 3.4 互动体系

#### 3.4.1 评论系统
- **两级嵌套**：一级评论 + 楼中楼回复（两层）
- **置顶**：作者可置顶最多一条一级评论，置顶评论排在最前
- **点赞**：可对评论点赞/取消点赞，点赞数实时更新
- **编辑窗口**：发布后 **10 分钟内**可修改评论内容，记录编辑次数
- **删除**：评论者和管理员均可删除，软删除保留记录
- **敏感词过滤**：评论内容自动扫描

#### 3.4.2 点赞与收藏
- 文章点赞 / 取消点赞（实时更新计数）
- 文章收藏到个人收藏夹，可查看收藏记录和收藏时间
- 点赞/收藏事件自动发送**站内通知**给文章作者

### 3.6 私信系统
