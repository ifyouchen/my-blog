# 博客网站第一版需求说明书

## 1. 项目背景

`my-blog` 是一个面向技术创作者和技术读者的博客社区。第一版目标不是完整复刻 CSDN，而是先完成一个可使用、可扩展、具备商业化雏形的技术博客 MVP。

系统采用前后端分离架构，核心能力包括用户注册登录、文章发布、文章浏览、评论互动、点赞收藏、个人中心和基础管理后台。

## 2. 项目目标

- 为用户提供稳定的技术文章发布和阅读体验。
- 为创作者提供文章管理、草稿编辑、数据反馈等基础能力。
- 为平台管理员提供用户、文章、评论和分类标签的基础管理能力。
- 为后续扩展专栏、付费内容、积分、推荐算法、资源下载等功能预留结构。

## 3. 用户角色

| 角色 | 说明 |
|---|---|
| 游客 | 未登录用户，可以浏览文章、搜索文章、查看评论。 |
| 普通用户 | 已登录用户，可以评论、点赞、收藏、维护个人资料。 |
| 作者 | 已登录且发布文章的用户，可以管理自己的文章。 |
| 管理员 | 平台管理人员，可以管理用户、文章、评论、分类和标签。 |

说明：第一版中普通用户和作者不需要单独注册角色。用户只要发布文章，即可具备作者行为。

## 4. 第一版功能范围

### 4.1 用户功能

- 用户注册。
- 用户登录。
- 用户退出。
- 获取当前登录用户信息。
- 修改头像、昵称、简介等个人资料。
- 查看用户个人主页。

### 4.2 文章功能

- 发布文章。
- 保存草稿。
- 编辑自己的文章。
- 删除自己的文章。
- 查看文章列表。
- 查看文章详情。
- 按标题、摘要、内容关键字搜索文章。
- 按分类和标签筛选文章。
- 记录文章阅读量。

### 4.3 分类和标签

- 文章发布时选择分类。
- 文章发布时添加标签。
- 首页和列表页支持按分类、标签筛选。
- 管理员可以维护分类和标签。

### 4.4 评论功能

- 登录用户可以评论文章。
- 登录用户可以回复评论。
- 用户可以删除自己的评论。
- 文章作者可以删除自己文章下的评论。
- 管理员可以删除任意评论。

### 4.5 点赞和收藏

- 登录用户可以点赞文章。
- 登录用户可以取消点赞。
- 登录用户可以收藏文章。
- 登录用户可以取消收藏。
- 用户可以查看自己的收藏文章列表。

### 4.6 创作者中心

- 查看我的文章列表。
- 按文章状态筛选：草稿、已发布、已下架、已删除。
- 编辑、删除、发布草稿文章。
- 查看文章基础数据：阅读量、点赞数、收藏数、评论数。

### 4.7 管理后台

- 管理员登录后进入后台。
- 查看用户列表。
- 禁用或启用用户。
- 查看文章列表。
- 下架或恢复文章。
- 查看评论列表。
- 删除评论。
- 管理分类和标签。
- 查看平台基础统计数据。

## 5. 第一版非目标范围

以下能力不纳入第一版必做范围：

- 付费文章。
- 付费专栏。
- 积分系统。
- 资源下载。
- 会员体系。
- 第三方登录。
- 个性化推荐算法。
- Elasticsearch 全文搜索。
- 移动 App。
- 小程序。
- 复杂运营活动。
- 多租户或企业空间。

## 6. 核心业务流程

### 6.1 注册登录流程

1. 用户进入注册页。
2. 输入用户名、邮箱、密码。
3. 后端校验用户名和邮箱是否可用。
4. 注册成功后返回用户基础信息。
5. 用户登录后，后端返回 JWT。
6. 前端保存 JWT，并在后续请求中放入 `Authorization` 请求头。

### 6.2 发布文章流程

1. 用户登录。
2. 进入发布文章页。
3. 填写标题、摘要、正文、分类、标签、封面图。
4. 选择保存草稿或发布文章。
5. 后端校验文章归属和字段合法性。
6. 保存成功后返回文章 ID。
7. 发布成功的文章进入文章列表和个人主页。

### 6.3 浏览文章流程

1. 用户进入首页或搜索结果页。
2. 系统展示已发布文章列表。
3. 用户点击文章进入详情页。
4. 后端返回文章正文、作者信息、分类标签、评论列表和互动状态。
5. 系统记录文章阅读量。

### 6.4 评论互动流程

1. 登录用户进入文章详情页。
2. 输入评论内容。
3. 后端校验登录状态、用户状态和评论内容。
4. 评论发布成功后更新文章评论数。
5. 用户可以点赞、收藏或取消对应互动。

### 6.5 后台审核流程

1. 管理员登录后台。
2. 查看文章、评论或用户列表。
3. 对违规文章执行下架。
4. 对违规评论执行删除。
5. 对异常用户执行禁用。
6. 系统记录管理员操作日志。

## 7. 数据范围

- 文章正文使用 Markdown 格式存储。
- 文章状态包括：草稿、已发布、已下架、已删除。
- 用户状态包括：正常、禁用、已删除。
- 文件上传第一版支持头像和文章封面。
- 阅读量第一版使用简单计数，不做复杂防刷策略。

## 8. 验收标准

- 用户可以完成注册、登录、退出和个人资料修改。
- 登录用户可以发布、编辑、删除自己的文章。
- 游客可以浏览首页、文章列表、文章详情和搜索结果。
- 登录用户可以评论、点赞、收藏文章。
- 用户不能编辑或删除别人的文章。
- 游客不能评论、点赞和收藏。
- 管理员可以管理用户、文章、评论、分类和标签。
- 被禁用用户不能登录、发文、评论、点赞和收藏。
- 所有受保护接口都需要校验 JWT。
- 第一版页面和接口可以支撑完整的博客阅读、写作和基础管理流程。
