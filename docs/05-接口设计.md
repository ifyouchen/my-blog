# 博客网站第一版接口设计

## 1. 接口规范

基础路径：`/api`

认证方式：JWT。

受保护接口需要请求头：

```http
Authorization: Bearer <token>
```

统一返回结构：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

分页返回结构：

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "items": [],
    "page": 1,
    "pageSize": 10,
    "total": 0
  }
}
```

## 2. 通用错误码

| 错误码 | 说明 |
|---:|---|
| `0` | 成功 |
| `400` | 请求参数错误 |
| `401` | 未登录或 Token 无效 |
| `403` | 无权限 |
| `404` | 资源不存在 |
| `409` | 数据冲突 |
| `500` | 服务器内部错误 |

## 3. 用户认证接口

### 3.1 注册

- 方法：`POST`
- 路径：`/api/auth/register`
- 权限：游客

请求体：

```json
{
  "username": "demo",
  "email": "demo@example.com",
  "password": "123456"
}
```

返回：

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1,
    "username": "demo",
    "email": "demo@example.com"
  }
}
```

### 3.2 登录

- 方法：`POST`
- 路径：`/api/auth/login`
- 权限：游客

请求体：

```json
{
  "account": "demo",
  "password": "123456"
}
```

返回：

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "jwt-token",
    "user": {
      "id": 1,
      "username": "demo",
      "nickname": "demo",
      "role": "USER"
    }
  }
}
```

### 3.3 退出登录

- 方法：`POST`
- 路径：`/api/auth/logout`
- 权限：登录用户

说明：JWT 模式下第一版可由前端删除本地 Token 完成退出，后端接口可保留用于后续黑名单机制。

### 3.4 获取当前用户

- 方法：`GET`
- 路径：`/api/users/me`
- 权限：登录用户

### 3.5 修改当前用户资料

- 方法：`PUT`
- 路径：`/api/users/me`
- 权限：登录用户

请求体：

```json
{
  "nickname": "新昵称",
  "avatarUrl": "/uploads/avatar.png",
  "bio": "热爱技术写作"
}
```

### 3.6 获取用户主页

- 方法：`GET`
- 路径：`/api/users/{id}`
- 权限：游客

## 4. 文章接口

### 4.1 获取文章列表

- 方法：`GET`
- 路径：`/api/articles`
- 权限：游客

查询参数：

| 参数 | 必填 | 说明 |
|---|---:|---|
| `page` | 否 | 页码，默认 1 |
| `pageSize` | 否 | 每页数量，默认 10 |
| `categoryId` | 否 | 分类 ID |
| `tagId` | 否 | 标签 ID |
| `keyword` | 否 | 搜索关键字 |
| `sort` | 否 | 排序方式：`latest`、`views`、`likes` |

返回字段：

```json
{
  "id": 1,
  "title": "Spring Boot 入门",
  "summary": "文章摘要",
  "coverUrl": "/uploads/cover.png",
  "author": {
    "id": 1,
    "nickname": "作者"
  },
  "category": {
    "id": 1,
    "name": "后端"
  },
  "tags": [
    {
      "id": 1,
      "name": "Spring Boot"
    }
  ],
  "viewCount": 100,
  "likeCount": 10,
  "commentCount": 3,
  "publishedAt": "2026-04-19 10:00:00"
}
```

### 4.2 获取文章详情

- 方法：`GET`
- 路径：`/api/articles/{id}`
- 权限：游客

说明：访问详情时可以增加阅读量。

### 4.3 发布文章

- 方法：`POST`
- 路径：`/api/articles`
- 权限：登录用户

请求体：

```json
{
  "title": "文章标题",
  "summary": "文章摘要",
  "content": "# Markdown 内容",
  "coverUrl": "/uploads/cover.png",
  "categoryId": 1,
  "tagIds": [1, 2],
  "status": "PUBLISHED"
}
```

### 4.4 编辑文章

- 方法：`PUT`
- 路径：`/api/articles/{id}`
- 权限：文章作者、管理员

### 4.5 删除文章

- 方法：`DELETE`
- 路径：`/api/articles/{id}`
- 权限：文章作者、管理员

说明：删除采用逻辑删除，文章状态更新为 `DELETED`。

### 4.6 获取我的文章

- 方法：`GET`
- 路径：`/api/users/me/articles`
- 权限：登录用户

查询参数：

| 参数 | 必填 | 说明 |
|---|---:|---|
| `status` | 否 | `DRAFT`、`PUBLISHED`、`OFFLINE`、`DELETED` |
| `page` | 否 | 页码 |
| `pageSize` | 否 | 每页数量 |

- 方法：`POST`
- 路径：`/api/admin/categories`
- 权限：管理员

{
  "name": "后端",

- 方法：`PUT`
- 路径：`/api/admin/categories/{id}`
- 权限：管理员

### 5.4 禁用或启用分类

- 方法：`PATCH`
- 路径：`/api/admin/categories/{id}/enabled`
- 权限：管理员

## 7. 评论接口

### 7.1 获取文章评论

- 方法：`GET`
- 路径：`/api/articles/{articleId}/comments`
- 权限：游客

### 7.2 发表评论

- 方法：`POST`
- 路径：`/api/articles/{articleId}/comments`
- 权限：登录用户

请求体：

```json
{
  "content": "评论内容",
  "parentId": 0
}
```

说明：`parentId` 为 `0` 表示顶级评论。

### 7.3 删除评论

- 方法：`DELETE`
- 路径：`/api/comments/{id}`
- 权限：评论作者、文章作者、管理员

## 9. 文件上传接口

### 9.1 上传图片

- 方法：`POST`
- 路径：`/api/uploads/images`
- 权限：登录用户
- 请求类型：`multipart/form-data`

返回：

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "url": "/uploads/2026/04/image.png"
  }
}
```
