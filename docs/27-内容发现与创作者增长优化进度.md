## 实施详情

### Task-1：首页双通道（推荐/关注）

**文件**：`frontend/src/views/HomeView.vue`

- 新增 `feedTab` 状态，支持 `recommend` / `following` 两种模式
- 通过 URL query `feedTab` 同步状态，刷新可恢复
- 登录用户默认「关注」，游客默认「推荐」
- 游客点击「关注」触发登录引导弹窗
- 接入 `getFollowingFeedApi` 调用关注流接口（已有 API）
- 推荐模式 fallback 到 `listArticlesApi`
- 新增 feed tab UI 按钮组件
- 埋点：`home_feed_tab_exposed`、`home_feed_tab_clicked`、`home_feed_list_loaded`

### Task-2：搜索无结果智能兜底

**文件**：`frontend/src/views/SearchView.vue`

- 新增 `isSearchEmpty` 判断：搜索意图存在 + 已加载 + 无结果
- 新增 `suggestKeywords` 计算逻辑：从热门词/分类/标签中模糊匹配，最多 3 个
- 新增 `hotTagsForEmpty`：热门标签最多 6 个
- 空态区域展示：相近关键词、热门标签、切换 tab 按钮
- 点击建议词触发 `applySuggestion` → 新搜索
- 切 tab 统一走 `switchSearchTab`
- 埋点：`search_empty_exposed`、`search_empty_suggestion_clicked`、`search_empty_switch_tab_clicked`

### Task-3：阅读完成触发推荐

**文件**：`frontend/src/views/ArticleDetailView.vue`

- 新增 `deepReadTriggered` 状态
- `handleScroll` 中判断 `readingProgress >= 70` 且未触发过 → 设为 true
- 渲染「继续阅读推荐条」：展示相关文章前 3 篇，含封面、分类、标题
- 点击推荐文章触发 `article_recommend_click` 埋点
- 仅触发一次，不遮挡正文与评论区
- 埋点：`article_deep_read_triggered`、`article_recommend_click`

### Task-4：Dashboard 任务中心

**文件**：`frontend/src/views/DashboardView.vue`

- 新增 `taskCards` 计算属性，规则化生成任务卡：
  - `create_first`：无文章 → 去写第一篇
  - `resume_draft`：草稿超 3 天未更新 → 继续写作
  - `publish_streak`：有草稿但从未发布 → 建议发布
- 最多显示 2 条，按优先级排序
- CTA 按钮跳转对应页面
- 埋点：`dashboard_task_exposed`、`dashboard_task_clicked`

### Task-5：埋点最小闭环
