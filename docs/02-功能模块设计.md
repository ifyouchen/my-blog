# my-blog 功能模块设计

## 1. 模块总览

| 模块 | 主要职责 |
|------|----------|
| 用户模块 | 注册、登录、密码管理、邮箱换绑、个人资料、个人主页 |
| 文章模块 | 文章 CRUD、版本管理、状态流转、导出、付费解锁 |
| 分类模块 | 分组分类体系 CRUD |
| 标签模块 | 标签 CRUD、热门标签 |
| 专栏模块 | 创作者专栏、文章编排、订阅 |
| 专题模块 | 平台专题、文章编排 |
| 评论模块 | 评论/回复 CRUD、点赞、置顶 |
| 点赞模块 | 文章/评论点赞、取消 |
| 收藏模块 | 文章收藏、查看列表 |
| 关注模块 | 关注/取消关注、关注动态 |
| 搜索模块 | 统一搜索、历史关键词 |
| 排行榜模块 | 文章排行、作者排行 |
| 推荐模块 | 文章推荐、专栏推荐、作者推荐 |
| 通知模块 | 事件通知、SSE 实时推送 |
| 私信模块 | 会话聊天、SSE 实时推送 |
| 文件上传模块 | 图片/文件上传、COS 直传 |
| 广告模块 | 广告位管理、投放管理、统计 |
| 公告模块 | 公告 CRUD、发布管控 |
| 敏感词模块 | WARN/BLOCK 两级 |
| 举报模块 | 内容举报、审核处理 |
| 邀请码模块 | 邀请码生成、使用 |
| 首页模块 | Bootstrap 聚合数据 |
| 创作者面板 | 数据概览、文章管理、专栏管理、成长面板 |
| 管理后台 | 全站管理、审计日志、数据导出 |
| 成长模块 | 经验/等级/积分/签到/徽章/邀请/解锁/分账 |

## 2. 用户模块

### 2.1 功能点

- 邮箱注册（需验证码）
- 邮箱/用户名 + 密码登录
- 退出登录
- 获取当前用户信息
- 修改个人资料（头像、昵称、简介、网站、GitHub、Twitter、所在地）
- 修改密码（旧密码验证）
- 换绑邮箱（新旧邮箱双重验证）
- 获取安全信息（登录记录）
- 查看用户公开主页（文章列表、统计）
- 管理员禁用/启用用户、更改角色
- 获取用户关注状态

### 2.2 业务规则

- 用户名和邮箱不能重复
- 密码 BCrypt 加密存储
- 用户被禁用后不能登录和操作
- 普通用户不能修改其他用户资料
- 管理员操作需记录 admin_log
- 注册需邮件验证码（IP + 邮箱双重频率限制）

## 3. 文章模块

### 3.1 功能点

- 创建文章（草稿/发布/定时发布）
- 编辑文章
- 更新文章状态（发布、下线、删除、恢复草稿）
- 获取文章列表（支持分类/标签/关键字/排序筛选）
- 获取文章详情（含解锁状态）
- 获取我的文章（按状态筛选）
- 获取用户文章列表（按热度）
- 文章版本管理（历史版本查看、回滚）
- 文章导出（ZIP 格式）
- 文章邻居查询（上下篇）
- 相关文章推荐
- 文章首页推荐申请
- 文章验证（发布前校验）
- 文章统计分析（创作者面板）
- 付费文章解锁设置

### 3.2 业务规则

- 只有登录用户可发布文章
- 文章作者和管理员可编辑/删除
- 普通用户不能编辑/删除他人文章
- 已删除文章不在前台展示
- 已下线文章仅作者和管理员可见
- 发布时标题和正文必填
- 阅读量在详情页访问时增加
- 文章状态流转校验：DRAFT → PUBLISHED → OFFLINE → DELETED

## 4. 分类与标签模块

### 4.1 分类

- 分类按组管理（category_group → category）
- 一篇文章可选择单个分类
- 分类名称唯一
- 管理员 CRUD

### 4.2 标签

- 一篇文章可绑定多个标签
- 标签名称唯一
- 支持热门标签展示
- 管理员 CRUD
- 标签使用计数

## 5. 专栏模块

### 5.1 功能点

- 创作者创建/编辑/删除专栏
- 专栏封面、简介、难度、来源配置
- 专栏内文章排序、章节、学习路径步骤
- 专栏推荐设置
- 专栏订阅/取消订阅
- 浏览公开专栏列表/详情
- 专栏内邻居文章查询

### 5.2 业务规则

- 专栏作者和管理员可管理专栏
- 专栏有 PUBLISHED/OFFLINE/DELETED 状态
- 文章可从专栏中移除（不删除文章本身）

## 6. 专题模块

### 6.1 功能点

- 平台创建/编辑/删除专题
- 专题封面、简介、难度、来源配置
- 专题内文章排序、章节、学习路径步骤
- 专题推荐设置
- 浏览公开专题列表/详情
- 专题内邻居文章查询
- 学习进度追踪

### 6.2 业务规则

- 专题由管理员创建维护
- 专题有 PUBLISHED/OFFLINE 状态

## 7. 评论模块

### 7.1 功能点

- 文章评论和回复（两级结构：根评论 → 回复）
- 编辑评论（有限次数）
- 删除评论
- 评论点赞/取消
- 评论置顶/取消置顶
- 评论分页查询

### 7.2 业务规则

- 游客可查看评论
- 登录用户可发表评论
- 评论作者可删除自己的评论
- 文章作者可删除自己文章下的评论
- 管理员可删除任意评论
- 软删除

## 8. 点赞模块

- 登录用户可点赞/取消点赞文章和评论
- 同一用户对同一资源只能点赞一次
- 已下线/已删除资源不能新增点赞

## 9. 收藏模块

- 登录用户可收藏/取消收藏文章
- 同一用户对同一文章只能收藏一次
- 可查看自己的收藏列表

## 10. 关注模块

### 10.1 功能点

- 关注/取消关注用户
- 查看粉丝列表
- 查看关注列表
- 查询关注状态
- 关注动态 Feed

### 10.2 业务规则

- 登录用户才能关注
- 不可关注自己

## 11. 搜索模块

### 11.1 功能点

- 统一搜索（文章、专栏、专题、标签、用户）
- 热门搜索关键词
- 近期搜索关键词（用户级别）
- 搜索 Bootstrap（聚合分类/标签数据）

### 11.2 业务规则

- 搜索使用 MySQL FULLTEXT INDEX（ngram parser）
- 只返回已发布文章
- 搜索范围包括标题、摘要、正文

## 12. 排行榜模块

- 文章排行榜（热门、最新、最受欢迎）
- 作者排行榜
- 支持按时间段/分类筛选

## 13. 推荐模块

- 首页推荐文章
- 推荐专栏
- 推荐作者
- 文章详情页相关推荐
- 基于标签的推荐

## 14. 通知模块

### 14.1 功能点

- 事件通知：文章被点赞、收藏、评论；评论被回复、点赞；被关注等
- 未读计数
- 通知列表
- 标记已读/全部已读
- SSE 实时推送

### 14.2 业务规则

- 通知接收者为被操作内容的作者或相关用户
- 已读/未读状态管理

## 15. 私信模块

### 15.1 功能点

- 创建会话
- 发送消息
- 消息列表
- 会话列表
- 删除会话
- 标记已读
- 未读计数
- SSE 实时推送

### 15.2 业务规则

- 双方参与者管理（participant_a_id < participant_b_id）
- 双方独立删除（a_deleted_at / b_deleted_at）

## 16. 文件上传模块

### 16.1 功能点

- 上传图片（头像、封面、文章图片）
- 上传文件
- COS 预签名 URL 直传
- 后端代理上传（开发环境降级）

### 16.2 业务规则

- 20MB/文件，22MB/请求
- 上传限流（IP + 时间窗口）
- 前端不接触 COS 密钥

## 17. 广告模块

- 广告位管理（slot_code 唯一）
- 广告投放 CRUD（标题、图片、链接、排期）
- 曝光/点击事件追踪
- 广告位数据统计

## 18. 公告模块

- 公告 CRUD
- 发布/取消发布
- 过期设置
- 目标用户筛选（ALL/AUTHOR/ADMIN）

## 19. 敏感词模块

- 敏感词 CRUD
- 级别：WARN（提示）/ BLOCK（屏蔽）
- 分类管理
- 客户端缓存 + 本地检测

## 20. 举报模块

- 举报文章/评论/用户
- 举报类型：SPAM/INFRINGEMENT/ABUSE/ILLEGAL/OTHER
- 管理员审核处理（驳回/下线/删除/禁用）

## 21. 首页模块

- Bootstrap 聚合数据（推荐文章、热门文章、分类、标签、排行榜等）
- 全站统计信息

## 22. 创作者面板（Dashboard）

### 22.1 功能点

- 数据概览：文章数、阅读量、点赞数、评论数、粉丝数
- 趋势图（7 日/30 日/90 日）
- 文章性能排行
- 互动分析
- 内容机会分析
- 文章管理（按状态筛选、编辑、删除）
- 收藏管理
- 专栏管理
- 成长面板（经验、等级、积分、徽章、签到）

## 23. 管理后台

### 23.1 功能点

- 全站统计面板
- 用户管理（搜索、禁用/启用、角色变更、推荐设置）
- 文章管理（搜索、分类/状态筛选、下架/恢复/精选/删除/批量操作、导出）
- 评论管理（搜索、审核/拒绝/删除）
- 举报处理（查看详情、驳回/下线/删除/禁用用户）
- 分类管理（分类组 + 分类 CRUD）
- 标签管理（标签 CRUD）
- 专栏管理（CRUD + 文章管理）
- 专题管理（CRUD + 文章管理）
- 广告管理（广告位 + 投放 CRUD）
- 公告管理（CRUD + 发布/取消发布）
- 敏感词管理（CRUD）
- 操作日志查看（按操作人/操作类型/时间筛选）
- 数据导出（文章/用户 CSV）
- 成长管理（经验规则、等级阈值、积分规则、签到奖励、等级奖励、积分账户查询、分账管理、推荐申请审核、年度创作者评选）

### 23.2 业务规则

- 只有管理员可访问
- 关键操作记录 admin_log（含操作前后快照）
- 所有管理操作在接口层校验 ADMIN 角色

## 24. 用户成长模块

详见 [经验与积分](./经验与积分/) 目录下的详细设计文档。

### 24.1 经验与等级

- 行为事件触发经验发放（LIKE/READ/COMMENT/FAVORITE/SHARE/FOLLOW/PUBLISH）
- 双角色发放（ACTOR + AUTHOR）
- 日上限控制（SKIP/PARTIAL）
- 8 级等级体系，每级对应经验阈值
- 不可降级

### 24.2 积分

- 可消费虚拟货币
- 来源：签到、邀请、充值、管理调分
- 消费：文章解锁
- 禁负策略
- 不可变流水审计

### 24.3 签到

- 连续签到阶梯奖励（10~60 积分）
- 断签重算
- 日历查询

### 24.4 邀请奖励

- 邀请注册奖励 50 积分
- 风控防刷（IP 频次、设备指纹、邀请人上限）

### 24.5 充值回调

- 1 元 = 1 积分（可配置）
- 回调验签、幂等防重

### 24.6 文章解锁

- 消耗积分解锁付费文章
- 同一用户对同一文章永久解锁
- 解锁订单、解锁关系、分账记录在同一事务完成

### 24.7 收益分账

- 平台:作者 = 50:50（可配置）
- 分账总和守恒验证
- 管理端分账流水查询

## 25. 模块关系

- 用户发布文章，文章归属于用户
- 文章关联一个分类、多个标签
- 评论/点赞/收藏/关注关联用户和目标资源
- 专栏/专题包含多篇文章
- 通知/私信关联用户和互动事件
- 成长模块通过 Spring 事件监听博客行为
- 管理员操作记录到 admin_log
