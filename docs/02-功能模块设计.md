# 博客网站第一版功能模块设计

## 1. 模块总览

系统第一版拆分为以下模块：

| 模块 | 主要职责 |
|---|---|
| 用户模块 | 注册、登录、退出、个人资料、个人主页。 |
| 文章模块 | 文章发布、编辑、删除、列表、详情、草稿和阅读量。 |
| 分类模块 | 文章分类维护和分类筛选。 |
| 标签模块 | 文章标签维护和标签筛选。 |
| 评论模块 | 评论、回复、删除评论。 |
| 点赞模块 | 点赞文章和取消点赞。 |
| 收藏模块 | 收藏文章、取消收藏、查看收藏列表。 |
| 搜索模块 | 文章关键字搜索和条件筛选。 |
| 文件上传模块 | 上传头像和文章封面。 |
| 后台管理模块 | 用户、文章、评论、分类、标签和基础统计管理。 |

## 2. 用户模块

### 2.1 功能点

- 用户注册。
- 用户登录。
- 用户退出。
- 获取当前用户信息。
- 修改个人资料。
- 查看用户主页。
- 管理员禁用或启用用户。

### 2.2 输入输出

| 操作 | 输入 | 输出 |
|---|---|---|
| 注册 | 用户名、邮箱、密码 | 用户 ID、用户名、邮箱、创建时间 |
| 登录 | 账号、密码 | JWT、用户基础信息 |
| 修改资料 | 头像、昵称、简介 | 更新后的用户资料 |
| 查看主页 | 用户 ID | 用户信息、文章列表、统计数据 |

### 2.3 业务规则

- 用户名和邮箱不能重复。
- 密码必须加密存储，不能明文保存。
- 用户被禁用后不能登录，也不能访问需要登录的业务接口。
- 普通用户不能修改其他用户资料。
- 管理员可以禁用或启用用户，但不能直接查看用户密码。

## 3. 文章模块

### 3.1 功能点

- 发布文章。
- 保存草稿。
- 编辑文章。
- 删除文章。
- 查看文章列表。
- 查看文章详情。
- 查看我的文章。
- 按分类、标签、关键字筛选文章。
- 统计阅读量、点赞数、收藏数、评论数。

### 3.2 输入输出

| 操作 | 输入 | 输出 |
|---|---|---|
| 发布文章 | 标题、摘要、正文、分类、标签、封面 | 文章 ID、文章状态 |
| 编辑文章 | 文章 ID、文章内容 | 更新结果 |
| 删除文章 | 文章 ID | 删除结果 |
| 文章列表 | 页码、分类、标签、关键字 | 分页文章列表 |
| 文章详情 | 文章 ID | 文章正文、作者、分类、标签、统计数据 |

### 3.3 业务规则

- 只有登录用户可以发布文章。
- 文章作者和管理员可以编辑或删除文章。
- 普通用户不能编辑或删除别人的文章。
- 已删除文章不在前台展示。
- 已下架文章不在前台公开展示，但作者和管理员仍可在管理入口查看。
- 文章发布时标题和正文必填。
- 阅读量在用户访问文章详情页时增加。

## 4. 分类模块

### 4.1 功能点

- 查询分类列表。
- 文章绑定分类。
- 管理员新增分类。
- 管理员编辑分类。
- 管理员禁用分类。

### 4.2 业务规则

- 一篇文章只能选择一个主分类。
- 分类名称不能重复。
- 被禁用分类不再出现在发布文章的可选列表中。
- 已有关联文章的分类不建议物理删除，使用禁用状态处理。

## 5. 标签模块

### 5.1 功能点

- 查询标签列表。
- 文章绑定多个标签。
- 管理员新增标签。
- 管理员编辑标签。
- 管理员禁用标签。

### 5.2 业务规则

- 一篇文章可以绑定多个标签。
- 标签名称不能重复。
- 标签用于文章筛选和搜索辅助。
- 被禁用标签不再出现在发布文章的推荐列表中。

## 6. 评论模块

### 6.1 功能点

- 查看文章评论。
- 发表评论。
- 回复评论。
- 删除评论。

### 6.2 输入输出

| 操作 | 输入 | 输出 |
|---|---|---|
| 获取评论 | 文章 ID、页码 | 评论分页列表 |
| 发表评论 | 文章 ID、评论内容 | 评论 ID、创建时间 |
| 回复评论 | 文章 ID、父评论 ID、评论内容 | 回复 ID、创建时间 |
| 删除评论 | 评论 ID | 删除结果 |

### 6.3 业务规则

- 游客可以查看评论，但不能发表评论。
- 评论内容不能为空。
- 评论作者可以删除自己的评论。
- 文章作者可以删除自己文章下的评论。
- 管理员可以删除任意评论。
- 评论删除采用逻辑删除。

## 7. 点赞模块

### 7.1 功能点

- 点赞文章。
- 取消点赞。
- 查询当前用户是否已点赞。
- 统计文章点赞数。

### 7.2 业务规则

- 登录用户才能点赞。
- 同一用户对同一文章只能点赞一次。
- 取消点赞后文章点赞数减少。
- 已删除或已下架文章不能新增点赞。

## 8. 收藏模块

### 8.1 功能点

- 收藏文章。
- 取消收藏。
- 查看我的收藏列表。
- 查询当前用户是否已收藏。
- 统计文章收藏数。

### 8.2 业务规则

- 登录用户才能收藏。
- 同一用户对同一文章只能收藏一次。
- 已删除或已下架文章不能新增收藏。
- 文章被删除后，不在用户收藏列表前台展示。

## 9. 搜索模块

### 9.1 功能点

- 按关键字搜索文章。
- 按分类筛选文章。
- 按标签筛选文章。
- 按发布时间、阅读量排序。

### 9.2 业务规则

- 第一版使用 MySQL `LIKE` 或基础条件查询实现。
- 搜索范围包括文章标题、摘要和正文。
- 只返回已发布文章。
- 复杂分词、高亮和相关性排序留到后续版本。

## 10. 文件上传模块

### 10.1 功能点

- 上传用户头像。
- 上传文章封面。
- 返回文件访问地址。

### 10.2 业务规则

- 第一版默认使用本地存储。
- 限制文件类型为常见图片格式。
- 限制文件大小，建议头像不超过 2MB，封面不超过 5MB。
- 后续可以替换为对象存储，不影响业务表结构。

## 11. 后台管理模块

### 11.1 功能点

- 后台首页基础统计。
- 用户管理。
- 文章管理。
- 评论管理。
- 分类管理。
- 标签管理。
- 管理员操作日志。

### 11.2 业务规则

- 只有管理员可以访问后台接口。
- 管理员下架文章后，前台不再公开展示该文章。
- 管理员禁用用户后，该用户不能登录和操作内容。
- 关键后台操作需要记录操作日志。

## 12. 模块关系

- 用户发布文章，文章归属于用户。
- 文章关联一个分类。
- 文章通过中间表关联多个标签。
- 评论归属于文章和用户。
- 点赞记录归属于文章和用户。
- 收藏记录归属于文章和用户。
- 后台管理操作由管理员用户发起，并记录到操作日志。
