# 博客网站第一版页面原型与页面清单

## 1. 说明

本文档描述第一版页面清单、页面元素、用户操作、权限限制和跳转关系。第一版先产出文字版原型，不包含图片线框图。

## 2. 页面总览

| 页面 | 路由建议 | 访问角色 |
|---|---|---|
| 首页 | `/` | 游客、登录用户 |
| 登录页 | `/login` | 游客 |
| 注册页 | `/register` | 游客 |
| 文章详情页 | `/articles/:id` | 游客、登录用户 |
| 发布文章页 | `/editor/new` | 登录用户 |
| 编辑文章页 | `/editor/:id` | 文章作者、管理员 |
| 个人主页 | `/users/:id` | 游客、登录用户 |
| 我的文章页 | `/dashboard/articles` | 登录用户 |
| 我的收藏页 | `/dashboard/favorites` | 登录用户 |
| 搜索结果页 | `/search` | 游客、登录用户 |
| 管理员后台首页 | `/admin` | 管理员 |
| 用户管理页 | `/admin/users` | 管理员 |
| 文章管理页 | `/admin/articles` | 管理员 |
| 评论管理页 | `/admin/comments` | 管理员 |
| 分类管理页 / 标签管理页 | `/admin/categories`、`/admin/tags`（兼容 `/admin/taxonomy` 跳转） | 管理员 |

## 3. 首页

### 3.1 页面内容

- 顶部导航：Logo、搜索框、登录/注册入口、写文章入口、用户头像。
- 分类导航：全部、前端、后端、数据库、人工智能、运维等。
- 文章列表：标题、摘要、作者、发布时间、分类、标签、阅读量、点赞数、评论数。
- 侧边区域：热门文章、热门标签、推荐作者。

### 3.2 用户操作

- 点击文章进入详情页。
- 输入关键字进入搜索结果页。
- 点击分类筛选文章。
- 点击标签筛选文章。
- 点击写文章，如果未登录则跳转登录页。

### 3.3 权限限制

- 游客可以浏览文章列表。
- 游客点击写文章、点赞、收藏等需要登录的入口时，跳转登录页。

## 4. 登录页

### 4.1 页面内容

- 账号输入框。
- 密码输入框。
- 登录按钮。
- 注册入口。
- 错误提示区域。

### 4.2 用户操作

- 输入账号和密码登录。
- 登录成功后跳转首页或来源页。
- 点击注册入口跳转注册页。

### 4.3 校验规则

- 账号不能为空。
- 密码不能为空。
- 登录失败时展示明确错误提示。

## 5. 注册页

### 5.1 页面内容

- 用户名输入框。
- 邮箱输入框。
- 密码输入框。
- 确认密码输入框。
- 注册按钮。
- 登录入口。

### 5.2 用户操作

- 输入信息提交注册。
- 注册成功后跳转登录页或自动登录进入首页。

### 5.3 校验规则

- 用户名不能为空。
- 邮箱格式必须正确。
- 密码和确认密码必须一致。
- 用户名和邮箱不能重复。

## 6. 文章详情页

### 6.1 页面内容

- 文章标题。
- 作者信息。
- 发布时间和更新时间。
- 分类和标签。
- 阅读量、点赞数、收藏数、评论数。
- Markdown 渲染后的正文。
- 代码高亮和代码复制入口。
- 点赞按钮。
- 收藏按钮。
- 评论列表。
- 评论输入框。
- 相关推荐文章。

### 6.2 用户操作

- 浏览文章内容。
- 点赞或取消点赞。
- 收藏或取消收藏。
- 发表评论。
- 回复评论。
- 删除有权限删除的评论。
- 点击作者进入个人主页。
- 点击标签进入搜索结果页。

### 6.3 权限限制

- 游客可以阅读文章和评论。
- 游客不能点赞、收藏、评论。
- 评论作者、文章作者和管理员可以删除评论。

## 7. 发布文章页

### 7.1 页面内容

- 标题输入框。
- 摘要输入框。
- Markdown 编辑器。
- 分类选择器。
- 标签选择器。
- 封面图上传。
- 保存草稿按钮。
- 发布按钮。

### 7.2 用户操作

- 编辑文章内容。
- 上传封面图。
- 保存草稿。
- 发布文章。

### 7.3 校验规则

- 发布文章时标题不能为空。
- 发布文章时正文不能为空。
- 发布文章时必须选择分类。
- 标签可以为空，但建议至少填写一个。

## 8. 编辑文章页

### 8.1 页面内容

与发布文章页一致，同时需要加载已有文章内容。

### 8.2 用户操作

- 修改文章标题、摘要、正文、分类、标签和封面。
- 保存为草稿。
- 更新并发布。

### 8.3 权限限制

- 文章作者可以编辑自己的文章。
- 管理员可以编辑或下架文章。
- 普通用户不能编辑别人的文章。

## 9. 个人主页

### 9.1 页面内容

- 用户头像。
- 昵称。
- 简介。
- 注册时间。
- 文章数量。
- 总阅读量。
- 用户文章列表。

### 9.2 用户操作

- 查看作者文章。
- 点击文章进入详情。
- 当前登录用户查看自己的主页时，可以进入个人资料编辑和我的文章页。

## 10. 我的文章页

### 10.1 页面内容

- 文章状态筛选：全部、草稿、已发布、已下架、已删除。
- 我的文章列表。
- 每篇文章展示标题、状态、阅读量、点赞数、评论数、更新时间。
- 操作按钮：编辑、删除、查看。

### 10.2 用户操作

- 查看自己的文章。
- 编辑文章。
- 删除文章。
- 跳转发布文章页。

## 11. 我的收藏页

### 11.1 页面内容

- 收藏文章列表。
- 文章标题、摘要、作者、收藏时间。
- 取消收藏按钮。

### 11.2 用户操作

- 查看收藏文章。
- 取消收藏。
- 点击文章进入详情页。

## 12. 搜索结果页

### 12.1 页面内容

- 搜索关键字。
- 分类筛选。
- 标签筛选。
- 排序方式：最新、最多阅读、最多点赞。
- 搜索结果列表。
- 空结果提示。

### 12.2 用户操作

- 修改搜索关键字。
- 切换分类、标签和排序条件。
- 点击文章进入详情页。

## 13. 管理员后台首页

### 13.1 页面内容

- 用户总数。
- 文章总数。
- 评论总数。
- 今日新增文章数。
- 今日新增用户数。
- 待处理内容概览。

### 13.2 用户操作

- 跳转用户管理。
- 跳转文章管理。
- 跳转评论管理。
- 跳转分类标签管理。

## 14. 用户管理页

### 14.1 页面内容

- 用户列表。
- 搜索用户。
- 用户状态筛选。
- 用户 ID、用户名、邮箱、状态、注册时间。
- 禁用或启用按钮。

### 14.2 权限限制

- 只有管理员可访问。

## 15. 文章管理页

### 15.1 页面内容

- 文章列表。
- 标题搜索。
- 分类筛选。
- 状态筛选。
- 作者、发布时间、阅读量、状态。
- 下架、恢复、删除操作。

### 15.2 权限限制

- 只有管理员可访问。

## 16. 评论管理页

### 16.1 页面内容

- 评论列表。
- 评论内容。
- 评论用户。
- 所属文章。
- 创建时间。
- 删除按钮。

### 16.2 权限限制

- 只有管理员可访问。

## 17. 分类标签管理页

### 17.1 页面内容

- 分类列表。
- 标签列表。
- 新增分类。
- 新增标签。
- 编辑名称。
- 禁用或启用。

### 17.2 权限限制

- 只有管理员可访问。
