## 2. 首页（`/`）

### 2.1 页面内容

- 顶部导航：Logo、搜索框、登录/注册入口、写文章入口、用户头像（登录后）、私信和通知图标（登录后）。
- 首页门户 Hero 区域（HomePortalHero）：展示精选文章 FeaturedArticlesStrip。
- 文章信息流（ArticleFeed）：综合推荐、最新发布、热门文章，滚动加载。
- 侧边栏（HomeSidebar）：作者排行榜、热门标签、推荐作者、推荐专栏、公告、广告。
- 专题展示（TopicStrip）。

### 2.2 用户操作

- 浏览文章列表，点击进入详情页。
- 输入关键字搜索。
- 点击分类/标签筛选文章。
- 关注作者、点赞、收藏（需登录）。
- 写文章入口（未登录时弹出登录弹窗）。

### 2.3 权限限制

- 所有内容对游客可见。
- 互动操作（点赞/收藏/关注/评论）需要登录。

## 3. 登录/注册页（`/login`、`/register`）

### 3.1 页面内容

- 登录/注册切换 Tab。
- 登录：账号（用户名或邮箱）+ 密码。
- 注册：用户名 + 邮箱 + 密码 + 确认密码，邮箱验证码。
- 忘记密码入口。

### 3.2 用户操作

- 登录成功后跳转来源页或首页。
- 注册成功后跳转首页。
- 支持邮箱验证码注册。

## 4. 文章详情页（`/articles/:id`）

### 4.1 页面内容

- 文章标题、作者信息（头像、昵称）、发布时间、分类、标签。
- 阅读量、点赞数、收藏数、评论数。
- TipTap/Markdown 渲染正文，代码高亮。
- 文章目录/大纲（ArticleToc）。
- 点赞/收藏按钮。
- 评论列表（CommentList），支持回复、置顶、点赞。
- 相关文章推荐、邻居文章导航。
- 文章锁态展示（ArticleLockBadge + ArticleUnlockModal）：需要积分解锁时才显示。

### 4.2 用户操作

- 点赞/取消点赞、收藏/取消收藏。
- 发表评论、回复评论、编辑评论（10分钟内）、点赞评论。
- 置顶/取消置顶评论（文章作者/管理员）。
- 删除有权限删除的评论。
- 关注作者。
- 积分解锁付费文章。

### 4.3 权限限制

- 游客可阅读文章和评论，不能互动。
- 评论作者可编辑（10分钟内）和删除自己的评论。
- 文章作者和管理员可删除任意评论、置顶评论。
- 文章作者和管理员可编辑文章。

## 5. 写文章/编辑文章页（`/editor/new`、`/editor/:id`）

### 5.1 页面内容

- 标题输入、摘要输入。
- 富文本编辑器（TipTap + 代码高亮插件 + Markdown 模式切换）。
- 分类选择、标签选择/输入。
- 封面图上传。
- 解锁设置（付费文章积分价格）。
- 保存草稿/发布按钮。

### 5.2 用户操作

- 富文本编辑，支持标题、加粗、斜体、引用、列表、代码块、表格、任务列表、图片、链接。
- 代码语法高亮（多语言）。
- 切换 Markdown 编辑模式。
- 封面图上传、文章内图片上传。
- 保存草稿、发布、下线。

### 5.3 权限限制

- 仅登录用户可访问。
- 编辑 `/:id` 校验当前用户为作者或管理员。

## 6. 个人主页（`/users/:id`）

### 6.1 页面内容

- 用户头像、昵称、简介、个人网站、GitHub、Twitter、所在地。
- 徽章展示（UserLevelBadge、BadgeWall）。
- 注册时间、文章数、总阅读量、粉丝数、关注数。
- 文章列表（按时间）、热门文章。
- 关注/取关按钮。

### 6.2 用户操作

- 查看文章列表。
- 关注/取关作者。

## 7. 创作者后台（Dashboard）

### 7.1 创作概览（`/dashboard/overview`）

- 数据统计：文章数、阅读量、点赞数、评论数、粉丝数。
- 趋势图：近 7 天/30 天阅读趋势。
- 最近互动和待处理项。

### 7.2 我的文章（`/dashboard/articles`）

- 文章列表：标题、状态、阅读量、点赞数、评论数、更新时间。
- 状态筛选：全部、草稿、已发布、已下线、已删除。
- 操作：编辑、查看、删除、调整状态。

### 7.3 我的收藏（`/dashboard/favorites`）

- 收藏文章列表：标题、摘要、作者、收藏时间。
- 取消收藏。

### 7.4 我的专栏（`/dashboard/columns`）

- 创建和管理专栏。
- 专栏文章排序、添加/移除文章。

### 7.5 经验积分（`/dashboard/growth`）

- 经验值、等级、升级进度条。
- 积分余额、积分流水。
- 每日签到、连续签到奖励。
- 徽章墙。
- 等级特权说明。

## 8. 个人设置（`/settings/profile`）

### 8.1 页面内容

- 头像上传。
- 昵称、简介、个人网站、GitHub、Twitter、所在地编辑。
- 修改密码。
- 换绑邮箱。

## 9. 内容发现页面

### 9.1 搜索页（`/search`）

- 关键字搜索 + 全文搜索（文章/用户/专栏）。
- 筛选：综合、文章、用户、专栏。
- 排序：最新、最多阅读、最多点赞。
- 搜索联想、热搜关键词、搜索历史。

### 9.2 发现页（`/explore`）

- 全站内容发现聚合页。

### 9.3 排行榜（`/ranking`）

- 文章排行榜（热门/最新）。
- 作者排行榜。

### 9.4 分类详情（`/categories/:id`）

- 分类下的文章列表。

### 9.5 标签详情（`/tags/:id`）

- 标签下的文章列表，热门标签。

## 10. 专栏与专题

### 10.1 专栏列表（`/columns`） / 专栏详情（`/columns/:id`）

- 专栏封面、标题、摘要、作者、文章数、订阅数。
- 专栏内文章列表（按学习路径排序）。
- 订阅/取消订阅专栏。

### 10.2 专题列表（`/topics`） / 专题详情（`/topics/:id`）

- 专题封面、标题、简介、文章数。
- 专题内文章列表。

## 11. 消息与通知

### 11.1 通知中心（`/notifications`）

- 系统通知：评论、点赞、收藏、关注、举报处理结果等。
- 未读/已读切换。
- 全部标记已读。
- SSE 实时推送新通知。

### 11.2 私信（`/messages/:conversationId?`）

- 基于会话的私信。
- 会话列表。
- 消息发送/接收，已读状态。
- SSE 实时推送新消息。

## 12. 阅读记录（`/history`）

### 12.1 页面内容

- 用户浏览过的文章列表（按时间倒序）。

## 13. 关注动态（`/following`）

### 13.1 页面内容

- 关注用户的文章动态流。

## 14. 管理后台（Admin）

### 14.1 布局

- 左侧导航菜单，右侧内容区。
- 菜单项：概览、用户、文章、评论、举报、分类、标签、专栏、专题、广告、公告、敏感词、管理员日志、经验积分。

### 14.2 后台概览（`/admin/overview`）

- 全站统计：用户总数、文章总数、评论总数、今日新增。
- 待处理：待审核评论、待处理举报。

### 14.3 用户管理（`/admin/users`）

- 用户列表：ID、用户名、邮箱、角色、状态、注册时间。
- 关键字搜索、状态筛选。
- 操作：启用/禁用、设置角色、推荐/取消推荐作者。

### 14.4 文章管理（`/admin/articles`）

- 文章列表：标题、作者、分类、状态、阅读量、发布时间。
- 关键字搜索、分类筛选、状态筛选。
- 操作：下架/恢复/精选/删除（支持批量）。

### 14.5 评论管理（`/admin/comments`）

- 评论列表：内容、用户、文章、时间。
- 审核通过/拒绝/删除。

### 14.6 举报管理（`/admin/reports`）

- 举报列表：举报人、目标类型、原因、状态。
- 处理：拒绝、下架文章、删除评论、禁用用户。

### 14.7 分类管理（`/admin/categories`）

- 分类列表 + 分类组管理。
- 增删改查、启用/禁用、排序。

### 14.8 标签管理（`/admin/tags`）

- 标签列表 + 标签组管理。
- 增删改查、启用/禁用。

### 14.9 专栏管理（`/admin/columns`）

- 全站专栏 CRUD，管理文章归属。

### 14.10 专题管理（`/admin/topics`）

- 全站专题 CRUD，管理文章归属。

### 14.11 广告管理（`/admin/ads`）

- 广告位配置 + 广告投放管理。
- 增删改查、启用/禁用、时间范围。

### 14.12 公告管理（`/admin/announcements`）

- 公告创建、编辑、发布、撤回。

### 14.13 敏感词管理（`/admin/sensitive-words`）

- 敏感词增删改查，设定警告或拦截级别。

### 14.14 管理员日志（`/admin/logs`）

- 管理员操作审计日志，按操作类型/结果状态/时间范围检索。

### 14.15 经验积分管理（`/admin/growth`）

- 经验规则配置、等级阈值管理。
- 积分规则配置、管理员调分。
- 签到奖励配置（连续/累计）。
- 等级奖励配置、特权配置。
- 用户积分账户查询、积分流水查看。
- 文章解锁分账流水查询、重试。
