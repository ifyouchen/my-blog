## 架构约束速查

```
同一工程，不同包：
  博客核心  →  com.myblog.*
  成长模块  →  com.myblog.growth.*

允许的跨模块依赖：
  成长模块 @EventListener 博客侧事件（单向依赖）
  成长模块 可注入 UserRepository 查用户基本信息

禁止的行为：
  ❌ 博客核心代码 import com.myblog.growth.*
  ❌ 成长模块直接调用博客的 AppService
  ❌ 跨模块直接操作对方的数据库表（读 blog_article 字段除外）
```
